# Data Analytics Portfolio

A collection of data analytics projects using SQL, Python, and visualization tools to answer real-world business questions.  
Each project focuses on clear problem definition, correct analysis, and business-relevant insights.

---

## Featured Projects

### Superstore Sales Analysis (Power BI)
- Built an interactive Power BI dashboard analyzing retail sales data from 2014–2017
- Created KPI cards for Total Sales, Total Profit, Total Orders, and Average Order Value (AOV)
- Performed time-series analysis using a correctly ordered Year–Month dimension to reveal seasonality
- Identified consistent Q4 sales spikes indicating strong year-end demand

**Tools:** Power BI, Power Query, DAX  
**Deliverables:** Interactive dashboard (.pbix) and static preview image  

[View project](projects/superstore-powerbi-analysis/)

---

### Amazon Sales Analysis (SQL & Python)
- Analyzed Amazon sales data using SQL to answer key business questions
- Performed data cleaning, aggregation, and exploratory analysis in Python
- Generated insights related to revenue trends, product performance, and customer behavior
- Combined SQL querying with Python-based analysis for reproducible results

**Tools:** SQL, Python, pandas, matplotlib  
**Deliverables:** SQL queries, Python analysis notebooks, and visualizations  

[View project](projects/amazon-sales-analysis/)
